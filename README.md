# Hello Ride – Carpooling Web Application

## Description

Hello Ride is a web-based carpooling (ride-sharing) application that enables users to share their travel with others.
Unlike traditional ride-booking systems, this platform allows a user with a vehicle to offer a ride, while other users can join the journey, making travel more **cost-efficient and collaborative**.

---

## Features

* Carpooling-based ride sharing (users can offer and join rides)
* Ride search based on pickup and destination
* Booking interface for joining available rides
* Google Maps integration for destination viewing
* Navigation between multiple sections (Home, Rides, Map, Profile)
* Dynamic profile display
